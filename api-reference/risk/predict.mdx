---
title: 'Predict T2DM Risk'
api: 'POST /api/v1/risk/predict'
description: 'Predict Type 2 Diabetes risk score from clinical patient data'
---

## Endpoint

```
POST https://api.t2dm-predictor.com/api/v1/risk/predict
```

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication
  ```
  Authorization: Bearer <your_api_token>
  ```
</ParamField>

## Request Body

<ParamField body="patient_id" type="string">
  Unique patient identifier (optional, for tracking)
</ParamField>

<ParamField body="age" type="integer" required>
  Patient age in years (18-120)
</ParamField>

<ParamField body="bmi" type="float" required>
  Body Mass Index (kg/m²) (15-60)
</ParamField>

<ParamField body="hba1c" type="float">
  Hemoglobin A1c percentage (4.0-15.0)
</ParamField>

<ParamField body="glucose_fasting" type="float">
  Fasting blood glucose (mg/dL) (50-400)
</ParamField>

<ParamField body="systolic_bp" type="integer">
  Systolic blood pressure (mmHg) (80-200)
</ParamField>

<ParamField body="diastolic_bp" type="integer">
  Diastolic blood pressure (mmHg) (40-130)
</ParamField>

<ParamField body="ldl" type="float">
  LDL cholesterol (mg/dL) (30-300)
</ParamField>

<ParamField body="hdl" type="float">
  HDL cholesterol (mg/dL) (20-100)
</ParamField>

<ParamField body="triglycerides" type="float">
  Triglycerides (mg/dL) (30-1000)
</ParamField>

<ParamField body="family_history" type="boolean">
  Family history of diabetes
</ParamField>

<ParamField body="smoking_status" type="string">
  Smoking status: "never", "former", "current"
</ParamField>

<ParamField body="physical_activity" type="string">
  Activity level: "sedentary", "light", "moderate", "vigorous"
</ParamField>

```python Python
import requests

url = "https://api.t2dm-predictor.com/api/v1/risk/predict"

headers = {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_TOKEN"
}

payload = {
    "patient_id": "PATIENT_12345",
    "age": 45,
    "bmi": 28.5,
    "hba1c": 5.9,
    "glucose_fasting": 110,
    "systolic_bp": 130,
    "diastolic_bp": 85,
    "ldl": 140,
    "hdl": 45,
    "triglycerides": 180,
    "family_history": True,
    "smoking_status": "former",
    "physical_activity": "light"
}

response = requests.post(url, json=payload, headers=headers)
result = response.json()

print(f"5-Year T2DM Risk: {result['risk_score']:.1%}")
print(f"Risk Category: {result['risk_category']}")
```

## Response

## Risk Categories

| Category | Risk Score | Description | Action |
|----------|------------|-------------|--------|
| **Low** | < 0.10 | Low risk of developing T2DM | Routine screening, healthy lifestyle |
| **Moderate** | 0.10 - 0.25 | Moderate risk, lifestyle modification beneficial | Enhanced screening, lifestyle intervention |
| **High** | 0.25 - 0.50 | High risk, aggressive prevention needed | Intensive lifestyle program, consider metformin |
| **Very High** | > 0.50 | Very high risk, medical intervention recommended | Medical management, frequent monitoring |
