---
title: 'Batch Risk Prediction'
api: 'POST /api/v1/risk/batch-predict'
description: 'Process multiple patient risk predictions simultaneously'
---

## Endpoint

```
POST https://api.t2dm-predictor.com/api/v1/risk/batch-predict
```

## Request

Submit array of patient records for batch processing.

## Response

Returns array of risk predictions.

<Note>
  Batch endpoint optimized for processing 100-10,000 patients.
  See [Predict](/api-reference/risk/predict) for single patient API.
</Note>
