## Endpoint

```
POST https://api.t2dm-predictor.com/api/v1/food/analyze
```

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication
</ParamField>

## Request Body

This endpoint accepts `multipart/form-data` for image upload.

<ParamField body="file" type="file" required>
  Food image file (JPEG, PNG, WEBP)
  - Max size: 10 MB
  - Recommended: High resolution, well-lit images
  - Supported formats: .jpg, .jpeg, .png, .webp
</ParamField>

<ParamField body="portion_reference" type="string">
  Optional reference object for portion estimation
  - "hand" - Use hand as reference
  - "plate" - Standard dinner plate
  - "coin" - Standard coin for scale
</ParamField>

<ParamField body="return_alternatives" type="boolean">
  Return top-5 predictions (default: false)
</ParamField>

## Request Example

<CodeGroup>
```bash cURL
curl -X POST https://api.t2dm-predictor.com/api/v1/food/analyze \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -F "file=@meal_photo.jpg" \
  -F "portion_reference=plate" \
  -F "return_alternatives=true"
```

```python Python
import requests

url = "https://api.t2dm-predictor.com/api/v1/food/analyze"

headers = {
    "Authorization": "Bearer YOUR_API_TOKEN"
}

files = {
    'file': open('meal_photo.jpg', 'rb')
}

data = {
    'portion_reference': 'plate',
    'return_alternatives': True
}

response = requests.post(url, headers=headers, files=files, data=data)
result = response.json()

print(f"Detected: {result['food_name']}")
print(f"Confidence: {result['confidence']:.1%}")
print(f"Calories: {result['nutrients']['calories']} kcal")
```

```javascript JavaScript
const FormData = require('form-data');
const fs = require('fs');
const axios = require('axios');

const form = new FormData();
form.append('file', fs.createReadStream('meal_photo.jpg'));
form.append('portion_reference', 'plate');
form.append('return_alternatives', 'true');

axios.post('https://api.t2dm-predictor.com/api/v1/food/analyze', form, {
  headers: {
    ...form.getHeaders(),
    'Authorization': 'Bearer YOUR_API_TOKEN'
  }
})
.then(response => {
  console.log(`Detected: ${response.data.food_name}`);
  console.log(`Confidence: ${(response.data.confidence * 100).toFixed(1)}%`);
  console.log(`Calories: ${response.data.nutrients.calories} kcal`);
})
.catch(error => console.error(error));
```

## Best Practices for Image Quality

<CardGroup cols={2}>
  <Card title="Lighting" icon="lightbulb">
    - Use natural or bright lighting
    - Avoid harsh shadows
    - No flash directly on food
  </Card>
  
  <Card title="Angle & Distance" icon="camera">
    - Top-down view preferred
    - Fill 60-80% of frame
    - 15-30 cm distance
  </Card>
  
  <Card title="Background" icon="image">
    - Plain, contrasting background
    - Avoid clutter
    - Use standard plate/bowl
  </Card>
  
  <Card title="Focus & Resolution" icon="magnifying-glass">
    - Sharp focus on food
    - Minimum 640x640 pixels
    - No excessive filters
  </Card>
</CardGroup>
