## Classification Metrics

- **AUROC**: Area under ROC curve
- **AUPRC**: Area under precision-recall curve
- **Sensitivity**: True positive rate
- **Specificity**: True negative rate
- **PPV/NPV**: Positive/negative predictive value

## Calibration Metrics

- **Brier Score**: Squared difference between predictions and outcomes
- **ECE**: Expected calibration error
- **Calibration Plots**: Predicted vs observed probabilities

## Computer Vision Metrics

- **Top-1/Top-5 Accuracy**: Classification accuracy
- **MAE/RMSE**: Nutrient estimation error
- **mAP**: Mean average precision (detection)
