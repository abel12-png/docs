## Alternative Architectures

Explore state-of-the-art tabular deep learning models:

- **TabNet**: Attention-based architecture
- **FT-Transformer**: Transformer for tabular data  
- **TabTransformer**: Contextual embeddings for categorical features
- **SAINT**: Self-Attention and Intersample Attention
