## Prerequisites

Before you begin, ensure you have the following:

<AccordionGroup>
  <Accordion title="System Requirements" icon="computer">
    - **OS**: Ubuntu 20.04+ / macOS 11+ / Windows 10+ with WSL2
    - **Python**: 3.9 or higher
    - **GPU**: NVIDIA GPU with 8GB+ VRAM (optional but recommended)
    - **RAM**: 16GB minimum, 32GB recommended
    - **Storage**: 100GB free space for datasets and models
  </Accordion>

  <Accordion title="Dataset Access" icon="key">
    - **MIMIC-IV**: Complete PhysioNet credentialing ([Apply here](https://physionet.org/settings/credentialing/))
    - **MM-Food-100K**: Request dataset access from the authors
    - Both datasets require signed data use agreements
  </Accordion>

  <Accordion title="Development Tools" icon="wrench">
    - Git
    - Docker (optional for containerized deployment)
    - CUDA Toolkit 11.8+ (for GPU support)
    - pip or conda package manager
  </Accordion>
</AccordionGroup>

## Installation

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/yourusername/t2dm-ai-predictor.git
    cd t2dm-ai-predictor
    ```
  </Step>

  <Step title="Create Virtual Environment">
    ```bash
    # Using conda (recommended)
    conda create -n t2dm python=3.10
    conda activate t2dm

    # Or using venv
    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    ```
  </Step>

  <Step title="Install Dependencies">
    ```bash
    # Install core dependencies
    pip install -r requirements.txt

    # Install development dependencies (optional)
    pip install -r requirements-dev.txt

    # Verify installation
    python -c "import torch; print(f'PyTorch: {torch.__version__}, CUDA: {torch.cuda.is_available()}')"
    ```
  </Step>
</Steps>

## Dataset Setup

### MIMIC-IV Clinical Data

<Steps>
  <Step title="Download MIMIC-IV">
    ```bash
    # Set up PhysioNet credentials
    export PHYSIONET_USERNAME="your_username"
    export PHYSIONET_PASSWORD="your_password"

    # Download MIMIC-IV core modules
    python scripts/data/download_mimic.py \
      --output-dir data/mimic-iv \
      --modules core hosp icu
    ```
  </Step>

  <Step title="Extract T2DM Cohort">
    ```bash
    # Filter patients with T2DM indicators
    python scripts/data/extract_t2dm_cohort.py \
      --mimic-dir data/mimic-iv \
      --output data/processed/t2dm_cohort.csv \
      --icd-codes configs/t2dm_icd_codes.json
    ```

    <Tip>
      The script identifies patients using ICD-10 codes: E11.* (Type 2 Diabetes), 
      E08.* (Diabetes due to underlying condition), and related complications.
    </Tip>
  </Step>

  <Step title="Preprocess Clinical Features">
    ```bash
    python scripts/data/preprocess_clinical.py \
      --cohort data/processed/t2dm_cohort.csv \
      --output data/processed/clinical_features.parquet \
      --include-labs --include-vitals --include-demographics
    ```
  </Step>
</Steps>

### MM-Food-100K Dataset

<Steps>
  <Step title="Download Food Dataset">
    ```bash
    # Download MM-Food-100K dataset
    python scripts/data/download_mmfood.py \
      --output-dir data/mm-food-100k \
      --split train val test

    # Expected structure:
    # data/mm-food-100k/
    # ├── images/
    # ├── annotations/
    # └── metadata.json
    ```
  </Step>

  <Step title="Verify Dataset Integrity">
    ```bash
    python scripts/data/verify_food_dataset.py \
      --data-dir data/mm-food-100k \
      --check-images --check-annotations
    ```
  </Step>
</Steps>

## Model Training

### Train Risk Prediction Model

```bash
# Train LSTM-based clinical risk model
python train_risk_model.py \
  --config configs/lstm_risk_config.yaml \
  --data data/processed/clinical_features.parquet \
  --output models/risk_lstm \
  --epochs 100 \
  --batch-size 64 \
  --early-stopping-patience 10
```

<CodeGroup>
```yaml configs/lstm_risk_config.yaml
model:
  type: "LSTM"
  hidden_size: 256
  num_layers: 3
  dropout: 0.3
  bidirectional: true

features:
  clinical:
    - age
    - bmi
    - hba1c
    - glucose_fasting
    - lipid_profile
  temporal:
    - vitals_sequence
    - lab_sequence
    
training:
  optimizer: "AdamW"
  learning_rate: 0.001
  weight_decay: 0.01
  scheduler: "CosineAnnealingLR"
  
loss:
  type: "BCEWithLogitsLoss"
  pos_weight: 2.5  # Handle class imbalance
```

### Train Food Recognition Model

```bash
# Train Vision Transformer for food classification
python train_food_model.py \
  --config configs/vit_food_config.yaml \
  --data data/mm-food-100k \
  --output models/food_vit \
  --epochs 50 \
  --batch-size 32 \
  --gpus 1
```

<CodeGroup>
```yaml configs/vit_food_config.yaml
model:
  type: "VisionTransformer"
  backbone: "vit_base_patch16_224"
  pretrained: true
  num_classes: 101
  
  # Multi-task heads
  tasks:
    - food_classification
    - portion_estimation
    - nutrient_regression

data:
  image_size: 224
  augmentation:
    - RandomHorizontalFlip
    - RandomRotation
    - ColorJitter
    - RandomResizedCrop
  
training:
  optimizer: "AdamW"
  learning_rate: 0.0001
  warmup_epochs: 5
  mixup_alpha: 0.2
```

## Running Inference

### Quick Prediction Example

```python predict_example.py
from t2dm_predictor import RiskPredictor, FoodAnalyzer
from PIL import Image

# Initialize models
risk_model = RiskPredictor.from_pretrained("models/risk_lstm")
food_model = FoodAnalyzer.from_pretrained("models/food_vit")

# Patient clinical data
patient_data = {
    "age": 45,
    "bmi": 28.5,
    "hba1c": 5.9,
    "glucose_fasting": 110,
    "systolic_bp": 130,
    "diastolic_bp": 85,
    "ldl": 140,
    "hdl": 45,
    "triglycerides": 180
}

# Predict T2DM risk
risk_score = risk_model.predict(patient_data)
print(f"5-Year T2DM Risk: {risk_score:.2%}")
print(f"Risk Category: {risk_model.get_risk_category(risk_score)}")

# Analyze food image
food_image = Image.open("sample_meal.jpg")
food_analysis = food_model.analyze(food_image)

print("\nFood Analysis:")
print(f"Detected: {food_analysis['food_name']}")
print(f"Confidence: {food_analysis['confidence']:.2%}")
print(f"\nNutrients:")
print(f"  Calories: {food_analysis['nutrients']['calories']} kcal")
print(f"  Protein: {food_analysis['nutrients']['protein']}g")
print(f"  Carbs: {food_analysis['nutrients']['carbs']}g")
print(f"  Fat: {food_analysis['nutrients']['fat']}g")
print(f"  Glycemic Index: {food_analysis['glycemic_index']}")

# Get personalized recommendations
from t2dm_predictor import RecommendationEngine

recommender = RecommendationEngine(risk_model, food_model)
recommendations = recommender.generate(
    patient_data=patient_data,
    current_meal=food_analysis
)

print("\n--- Personalized Recommendations ---")
for rec in recommendations:
    print(f"• {rec}")
```


## Launch API Server

<Steps>
  <Step title="Start the API">
    ```bash
    # Development mode
    uvicorn api.main:app --reload --host 0.0.0.0 --port 8000

    # Production mode with workers
    gunicorn api.main:app \
      --workers 4 \
      --worker-class uvicorn.workers.UvicornWorker \
      --bind 0.0.0.0:8000
    ```
  </Step>

  <Step title="Test the Endpoints">
    ```bash
    # Health check
    curl http://localhost:8000/health

    # Predict T2DM risk
    curl -X POST http://localhost:8000/api/v1/risk/predict \
      -H "Content-Type: application/json" \
      -d '{
        "age": 45,
        "bmi": 28.5,
        "hba1c": 5.9,
        "glucose_fasting": 110
      }'

    # Analyze food image
    curl -X POST http://localhost:8000/api/v1/food/analyze \
      -F "file=@sample_meal.jpg"
    ```
  </Step>

  <Step title="View API Documentation">
    Open your browser and navigate to:
    - Swagger UI: http://localhost:8000/docs
    - ReDoc: http://localhost:8000/redoc
  </Step>
</Steps>

## Docker Deployment

```bash
# Build Docker image
docker build -t t2dm-predictor:latest .

# Run container with GPU support
docker run --gpus all \
  -p 8000:8000 \
  -v $(pwd)/models:/app/models \
  -v $(pwd)/data:/app/data \
  t2dm-predictor:latest

# Or use Docker Compose
docker-compose up -d
```

<CodeGroup>
```dockerfile Dockerfile
FROM nvidia/cuda:11.8.0-cudnn8-runtime-ubuntu22.04

WORKDIR /app

# Install Python
RUN apt-get update && apt-get install -y python3.10 python3-pip

# Copy requirements
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# Copy application
COPY . .

# Expose API port
EXPOSE 8000

# Health check
HEALTHCHECK --interval=30s --timeout=10s --start-period=40s --retries=3 \
  CMD curl -f http://localhost:8000/health || exit 1

CMD ["uvicorn", "api.main:app", "--host", "0.0.0.0", "--port", "8000"]
```

